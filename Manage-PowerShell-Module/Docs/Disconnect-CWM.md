# Disconnect-CWM

## SYNOPSIS
This will remove the ConnectWise Manage authorization variable.

## SYNTAX

```
Disconnect-CWM [<CommonParameters>]
```

## DESCRIPTION
This will remove the ConnectWise Manage authorization variable.

## EXAMPLES

### EXAMPLE 1
```powershell
Disconnect-CWM
```
